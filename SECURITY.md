# Security Policy

## Supported Versions

Currently, only the latest version of MARS Studio is actively supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Legacy  | :x:                |

## Reporting a Vulnerability

Because MARS Studio is an educational desktop application running locally, traditional web vulnerabilities are less common. However, if you discover an exploit (e.g., related to the C-compiler execution or local file system access), please do not report it via public GitHub issues.

Instead, please send a direct message or email.

We will acknowledge receipt of your vulnerability report as soon as possible and strive to send you regular updates about our progress in fixing it.
